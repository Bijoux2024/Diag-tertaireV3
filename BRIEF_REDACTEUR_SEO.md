# Brief Rédacteur Web / SEO - DiagTertiaire

> Ce document est la source de vérité pour toute rédaction liée à diag-tertiaire.fr.
> Il garantit la cohérence de script entre le site web et les articles de blog.

---

## 1. Identité du service

**Nom exact** : DiagTertiaire (pas "Diag Tertiaire", pas "diag-tertiaire")
**URL** : diag-tertiaire.fr
**Nature** : Outil en ligne de pré-diagnostic énergétique pour bâtiments tertiaires
**Cible** : Propriétaires, exploitants et bailleurs de locaux tertiaires de moins de 1 000 m²
**Positionnement** : Outil de cadrage rapide, gratuit, avant de décider si un audit complet est nécessaire

---

## 2. Pitch de référence (script officiel)

### Phrase signature (H1 du site)
> Votre bâtiment tertiaire consomme-t-il trop ?

### Description courte (sous-titre)
> Comparez votre consommation aux références de votre secteur. Résultat gratuit en PDF, en 3 minutes.

### Description longue (pour les présentations étoffées)
> DiagTertiaire fournit une estimation claire, sans rendez-vous ni démarche complexe.
> Quelques informations suffisent pour décider.
> Le tertiaire a ses propres repères. Trois informations concrètes pour savoir quoi faire ensuite.

### Phrase de réassurance
> Pour les locaux tertiaires de moins de 1 000 m².

### Phrase de clôture / relance
> Prêt à comparer votre consommation ?
> Une estimation concrète avant d'engager du temps ou du budget.

---

## 3. CTA principal

### Libellé exact
> **Lancer le pré-diagnostic gratuit**

### Sous-CTA (micro-copy)
> Sans engagement. Rapport PDF immédiat.

### Regles CTA
- Toujours utiliser "Lancer le pré-diagnostic gratuit" (jamais "Lancer mon diagnostic", "Tester", "Essayer")
- Le sous-CTA ne mentionne plus "3 minutes" (c'est déjà dans le sous-titre)
- Ne jamais ajouter de superlatifs ("le meilleur", "le plus rapide")

---

## 4. Vocabulaire autorise / interdit

### Mots-cles a utiliser
| Utiliser | Contexte |
|---|---|
| pré-diagnostic | Nature du service (pas "diagnostic" seul) |
| estimation | Ce que l'outil produit ("estimation claire", "estimation fiable", "estimation concrète") |
| comparer / comparaison | Action principale de l'utilisateur |
| références sectorielles | Base de calcul |
| rapport PDF | Livrable |
| cadrage | Positionnement du service (aide a la decision, pas audit reglementaire) |
| décider / décision | Bénéfice utilisateur |
| gratuit | Uniquement en contexte factuel, jamais en accroche isolée |

### Formulations interdites
| Ne pas utiliser | Pourquoi |
|---|---|
| diagnostic (seul, sans "pré-") | Confusion avec audit réglementaire officiel |
| audit | DiagTertiaire n'est pas un audit, c'est un pré-diagnostic |
| tiret cadratin (—) | Remplacé par une virgule, un point ou un tiret court (-) |
| "le meilleur", "ultra", "révolutionnaire" | Pas de marketing excessif |
| SLA, account manager, onboarding | Anglicismes a franciser |
| upgrade, downgrade | Dire "passer a l'offre supérieure" / "revenir a l'offre précédente" |
| Starter, Enterprise | Noms d'offres : "Essentiel" et "Sur mesure" |
| "en seulement X minutes" | Redondant si déjà mentionné |
| faux témoignages avec noms inventés | Utiliser des descriptions factuelles |

---

## 5. Ton editorial

- **Francais simple** : phrases courtes, pas de jargon technique inutile
- **Factuel** : pas de promesses, pas d'exagérations, pas de superlatifs marketing
- **Direct** : sujet + verbe + complément, aller droit au point
- **Sobre** : pas d'emojis dans le corps de texte (sauf contexte réseau social)
- **Pas de redondance** : ne pas répéter "gratuit" et "3 minutes" dans la meme section

---

## 6. Application dans les articles de blog

### Ce qui DOIT etre coherent avec le site
- La **section de présentation de DiagTertiaire** en fin d'article (encadré, bio, CTA)
- Le **CTA** exact : "Lancer le pré-diagnostic gratuit"
- La **description du service** : reprendre le pitch court ou long ci-dessus
- Le **positionnement** : outil de cadrage avant audit, pas un audit lui-meme

### Ce qui est LIBRE dans les articles
- Le titre de l'article (optimisé SEO, ton éditorial libre)
- Le corps de l'article (vulgarisation, ton pédagogique autorisé)
- Les exemples, chiffres, cas pratiques
- Le vocabulaire technique du secteur (décret tertiaire, DPE, ECS, etc.)

### Template de bloc CTA pour les articles

```html
<!-- Bloc CTA DiagTertiaire - a insérer en fin d'article -->
<div class="cta-blog">
  <h3>Votre batiment tertiaire consomme-t-il trop ?</h3>
  <p>Comparez votre consommation aux références de votre secteur.
     Résultat gratuit en PDF, en 3 minutes.</p>
  <a href="https://diag-tertiaire.fr/#diagnostic">Lancer le pré-diagnostic gratuit</a>
  <small>Sans engagement. Rapport PDF immédiat.</small>
</div>
```

### Template de mention courte (dans le corps d'un article)
> DiagTertiaire est un outil gratuit de pré-diagnostic énergétique pour les locaux tertiaires de moins de 1 000 m². Il compare votre consommation aux références de votre secteur et génère un rapport PDF en quelques minutes.
